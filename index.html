다가오는 경기

이용 방법

  1. 일정 탭에서 전체 경기 일정을 확인하세요.
  2. 내 예측 탭에서 닉네임을 입력하고 각 경기의 예상 스코어를 입력하세요.
  3. "브라우저에 저장"으로 임시 저장하거나, "GitHub에 제출"로 예측을 영구 저장하세요. (GitHub 계정 필요)
  4. 경기 결과가 업데이트되면 조별 순위와 랭킹이 자동으로 계산됩니다.

경기 결과가 입력되면 자동으로 순위가 계산됩니다. 현재 모든 경기가 미진행 상태이면 모든 팀이 0점입니다.

내 정보

닉네임은 예측 순위표에 표시됩니다. 같은 GitHub 계정으로 다시 제출하면 이전 예측을 덮어씁니다.

경기 예측 입력 (조별리그)

예측을 입력하지 않은 경기는 제출 시 제외됩니다. 토너먼트 대진은 조별리그 결과가 확정된 후 추가됩니다.

"GitHub에 제출"을 누르면 예측 내용이 담긴 GitHub 이슈 작성 화면이 열립니다. GitHub 계정으로 로그인 후 이슈를 등록(Submit new issue)하면, 자동화 작업이 예측을 data/tips.json에 저장합니다.

채점 방식

  • 4점 – 정확한 스코어를 맞춘 경우
  • 3점 – 골득실차는 맞췄지만 스코어가 다른 경우
  • 2점 – 승/무/패 결과만 맞춘 경우
  • 0점 – 결과를 맞추지 못한 경우
